# React-redux-mysql
    
    
   准备实现一个资源上传和交易网站，后期继续完善。
   使用react+redux实现前端的页面和逻辑；后台用到express框架，数据库使用的是mysql，之前有用过mongodb的，都是可以的；
   其中还有登录的浏览器存储，这里使用的不是cookie和session，而是用的JWT（json web token）。
